
#. Dada a frase ``Python é muito legal.``, use fatiamento para dar nome às
   variáveis contendo cada palavra. O resultado final deve ser:

   .. code-block:: python3

      >>> frase = "Python é muito legal."
      # resolução do problema aqui
      >>> palavra1
      "Python"
      >>> palavra2
      "é"
      >>> palavra3
      "muito"
      >>> palavra4
      "legal"
